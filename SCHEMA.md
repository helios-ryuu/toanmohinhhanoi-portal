# Database Schema (Supabase PostgreSQL)

Tai lieu nay tong hop schema hien tai cua du an, phu hop de chay tren Supabase SQL Editor.

## 1) Tong quan bang

- `author`: thong tin tac gia bai viet
- `tag`: nhan phan loai bai viet
- `series`: chuoi bai viet
- `post`: bai viet chinh
- `post_tags`: bang quan he nhieu-nhieu giua post va tag
- `user`: tai khoan nguoi dung cho luong auth tu xay dung (khong phai Supabase Auth)

## 2) Quan he giua cac bang

- `post.author_id -> author.id` (many-to-one)
- `post.series_id -> series.id` (many-to-one)
- `post_tags.post_id -> post.id` (many-to-one)
- `post_tags.tag_id -> tag.id` (many-to-one)

## 3) SQL DDL de khoi tao schema

Chay block SQL ben duoi trong Supabase SQL Editor.

```sql
-- =====================================================================
-- Core tables
-- =====================================================================

create table if not exists public.author (
    id bigint generated by default as identity primary key,
    name text not null unique,
    title text not null,
    avatar_url text,
    github_url text,
    linkedin_url text,
    created_at timestamptz not null default now()
);

create table if not exists public.tag (
    id bigint generated by default as identity primary key,
    name text not null unique,
    slug text not null unique,
    created_at timestamptz not null default now(),
    constraint tag_name_len_check check (char_length(name) <= 15)
);

create table if not exists public.series (
    id bigint generated by default as identity primary key,
    name text not null unique,
    slug text not null unique,
    description text,
    created_at timestamptz not null default now()
);

create table if not exists public.post (
    id bigint generated by default as identity primary key,
    slug text not null unique,
    title text not null,
    description text not null,
    content text not null,
    image_url text not null,
    level text not null,
    type text not null default 'standalone',
    series_id bigint references public.series(id) on delete set null,
    series_order integer,
    author_id bigint references public.author(id) on delete set null,
    reading_time text not null,
    published boolean not null default false,
    published_at timestamptz,
    created_at timestamptz not null default now(),
    updated_at timestamptz,

    constraint post_level_check
        check (level in ('beginner', 'intermediate', 'advanced')),

    constraint post_type_check
        check (type in ('standalone', 'series')),

    constraint post_series_consistency_check
        check (
            (type = 'standalone' and series_id is null and series_order is null)
            or
            (type = 'series' and series_id is not null and series_order is not null and series_order > 0)
        ),

    constraint post_publish_consistency_check
        check (
            (published = false and published_at is null)
            or
            (published = true and published_at is not null)
        )
);

create table if not exists public.post_tags (
    post_id bigint not null references public.post(id) on delete cascade,
    tag_id bigint not null references public.tag(id) on delete cascade,
    created_at timestamptz not null default now(),
    primary key (post_id, tag_id)
);

-- "user" la tu khoa he thong, vi vay can dat trong dau nhay kep
create table if not exists public."user" (
    id bigint generated by default as identity primary key,
    email text not null unique,
    phone text unique,
    username text not null unique,
    password_hash text not null,
    avatar_url text,
    display_name text,
    created_at timestamptz not null default now(),
    updated_at timestamptz,

    constraint user_username_len_check
        check (char_length(username) between 3 and 30),

    constraint user_username_format_check
        check (username ~ '^[A-Za-z0-9_]+$')
);

-- =====================================================================
-- Indexes
-- =====================================================================

create index if not exists idx_post_published_dates
    on public.post (published, published_at desc, created_at desc);

create index if not exists idx_post_author_id
    on public.post (author_id);

create index if not exists idx_post_series_id
    on public.post (series_id);

create index if not exists idx_post_tags_tag_id
    on public.post_tags (tag_id);

create index if not exists idx_user_created_at
    on public."user" (created_at desc);

-- =====================================================================
-- updated_at trigger helper
-- =====================================================================

create or replace function public.set_updated_at()
returns trigger
language plpgsql
as $$
begin
    new.updated_at = now();
    return new;
end;
$$;

drop trigger if exists trg_post_set_updated_at on public.post;
create trigger trg_post_set_updated_at
before update on public.post
for each row
execute function public.set_updated_at();

drop trigger if exists trg_user_set_updated_at on public."user";
create trigger trg_user_set_updated_at
before update on public."user"
for each row
execute function public.set_updated_at();
```

## 4) Ghi chu cho Supabase

- Du an hien dang truy cap DB qua server-side `DATABASE_URL` (package `postgres`), khong query truc tiep bang anon key.
- Neu ban co ke hoach query truc tiep tu client (Supabase JS), hay bat RLS va viet policy cho tung bang.
- Neu chi dung server route nhu hien tai, co the giu RLS tat cho bo bang tren (tuy mo hinh bao mat cua ban).
